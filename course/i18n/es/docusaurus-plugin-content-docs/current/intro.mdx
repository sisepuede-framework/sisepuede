# Bienvenida al curso de SISEPUEDE

<CourseHero style={{maxWidth: '100%', height: 'auto', marginBottom: '1.5rem'}} />

Este curso te enseña a usar **SISEPUEDE** (SImulando SEndas sectoriales y exploración de incertidumbre para la DEscarbonización) — un marco integrado en Python/Julia para el análisis exploratorio de trayectorias de descarbonización a nivel país bajo incertidumbre profunda. Aprenderás la arquitectura del modelo, su esquema de variables, cada modelo sectorial de emisiones, cómo se componen las transformaciones y estrategias de política, y cómo ejecutar experimentos DMDU completos.

## ¿A quién va dirigido este curso?

A personas investigadoras, analistas y tomadoras de decisiones que necesiten construir, ejecutar o interpretar escenarios de SISEPUEDE — incluyendo equipos técnicos de ministerios, centros de pensamiento, bancos de desarrollo y universidades que trabajan en estrategias de mitigación de largo plazo, análisis de NDC o contabilidad del Artículo 6.

## Cómo usar este curso

Las Partes I y II son prerrequisitos para el resto del material. Si es tu primer acercamiento a SISEPUEDE, léelas en orden. Las Partes III a V pueden consultarse de manera independiente una vez que domines el esquema. La Parte VI contiene notebooks ejecutables — úsalos para consolidar los conceptos de las partes anteriores.
